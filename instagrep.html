02 — Features

8 Tabs, One Interface

Every tab serves a distinct intelligence function. From raw profile data to behavioral pattern analysis to full media downloads — organized, structured, and exportable.

🖼️
Profile Picture
HD avatar with neon ring and gradient glow effect. Neon ring uses gradient animation cycling cyan → purple → pink.
📊
Animated Stats
Follower, following, and post counts animate with count-up effect on load. Engagement rate calculated and graded (Poor / Average / Good / Excellent).
✅
Account Intelligence
Account type (Public/Private/Business), verification badge, business category, external URL, full biography with auto-extracted emails, phones, and @mentions.
💡
Bio Intelligence
Regex-powered extraction from biography text: email addresses, phone numbers, URLs, @mentions, hashtags — all highlighted and clickable.
📧
Email Extraction
Business email from profile API + regex scan of all 50 post captions. Deduplicates and lists all unique emails found.
📞
Phone Numbers
Business phone from profile data + pattern matching across post captions. Handles international formats.
📍
Physical Address
Business address extracted from Instagram's business profile API. Shown only for accounts with public business information.
🔗
URL & Mentions
All external URLs from bio and posts collected. @mentions catalogued. Builds a network map of connected accounts.
⏰
Posting Time Analysis
Peak posting hour (0-23) with annotation. Peak posting day (Mon-Sun). Average posts per day calculated from timestamp history.
📈
Engagement Metrics
Average likes and comments per post. Engagement rate vs followers. Trend analysis over last 50 posts — is engagement growing or declining?
#️⃣
Hashtag Intelligence
Top 20 hashtags ranked by frequency. Shows exactly which tags get the most engagement. Horizontal bar chart with unique colors per hashtag.
📡
Live Status
Active stories indicator. Live broadcast status. Last post timestamp. Last story timestamp. Real-time presence detection.
🖼️
Profile Picture HD
Downloads the full-resolution profile picture. Saved to downloads/[username]/Profile/profile_pic.jpg.
📦
Bulk Post Download
Downloads all images and videos from the last 50 posts. Organized by post ID. Captions saved as caption.txt alongside media files.
⏱️
Progress Tracking
Live progress bar with percentage. Per-file status log. Cancel button to stop mid-download. File size and completion status shown.
📁
Organized Structure
downloads/[username]/Profile/ and Posts/post_[id]/ folder hierarchy created automatically. Clean, navigable output.
📄
JSON Export
Complete raw data dump for developers. All fields, all posts, all extracted intel in a single structured JSON file.
📊
Excel Workbook
Multi-sheet workbook: Profile, Posts, Hashtags, Locations. Styled with headers, alternating rows, and column formatting via openpyxl.
🌐
HTML Dashboard
Interactive dark-themed HTML report. Standalone file — open in any browser. Charts embedded, fully formatted, shareable.
📋
PDF Report
Professional PDF with embedded charts, profile summary, post table, and executive-style formatting via reportlab.

05 — Development Journey

Built in One Day

From concept to published GitHub repository. 3000+ lines written, 7 bugs fixed, 4 charts built, and 4 export formats implemented — all on March 4, 2026.

Phase 1 — Planning
Concept & Feature Definition
Decided the core goal: Instagram OSINT with GUI, no password required. Identified 8 core modules — profile, contact, locations, posts, activity, downloads, export, raw data. Chose cookie-based authentication as the only way to bypass Instagram's bot detection reliably.
Phase 2 — Environment
Kali Linux Setup
Created ~/projects/Instagrep, set up Python virtual environment, installed all 10 libraries. Configured .gitignore to exclude cookies.json and the downloads folder from version control.
Phase 3 — Core GUI
CustomTkinter Interface
Built the full 8-tab GUI with dark theme. Designed the 40+ color palette (deep space background, neon cyan accents, cyber green). Added glass-morphism card effects. Built the neon ring avatar with gradient animation and the animated stat counters.
Phase 4 — Data Engine
instagrapi Integration
Implemented cookie-based session management using browser-cookie3 to auto-extract cookies from Chrome, Firefox, Edge, and Brave. Built the retry engine with exponential backoff for 429 rate limit handling. Extracted 30+ data fields per profile.
Phase 5 — Charts
matplotlib Visualization Engine
Built 4 professional charts: hourly activity (24 bars), weekly activity (7 bars), top hashtags (horizontal), engagement over time (line with fill). Added peak annotations, average lines, gradient fills, and dual Y-axis for the engagement chart.
Phase 6 — Export
4 Export Formats
JSON raw dump, Excel multi-sheet workbook with styling, standalone HTML dashboard, and PDF report with embedded charts. Each format handles its own ANSI stripping, image embedding, and file naming.
Phase 7 — Debugging
7 Bugs Fixed
Fixed ImageTk import for canvas images, hex colors with alpha channels (not supported in tkinter), cookie_dict setter issue in instagrapi, 429 rate limit handling with backoff, LoginRequired session refresh, git push rejection, and README merge conflict resolution.
Phase 8 — Published
GitHub Release
Initialized git repository, created instagrep.py, README.md, requirements.txt, .gitignore, LICENSE (MIT). Force-pushed to handle existing README from GitHub repo creation. Project live at github.com/Jishnuec/InstaGrep.
